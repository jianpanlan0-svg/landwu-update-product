# 上传器 v1

## 当前结构

- 本地服务：负责文件夹选择、任务执行、鉴权保存
- 本地网页：负责可视化操作
- 脚本猫桥接：可选，用来把当前网页登录态同步到本地服务

## 启动

- 启动文件：`C:\Users\Administrator\Documents\Playground\领物TEMU上传器_v2.0.3\启动领物TEMU上传器_v2.0.3.bat`
- 本地页面：`http://127.0.0.1:18321`

## 推荐用法

1. 启动本地上传器
2. 在脚本猫安装桥接脚本  
   `C:\Users\Administrator\Documents\Playground\领物TEMU上传器_v2.0.3\userscripts\landwu_桥接同步登录态_双兼容_v5.user.js`
3. 打开任意 `user.landwu.com` 页面
4. 桥接脚本会自动把当前账号同步到本地上传器
5. 回到本地上传器，确认状态显示 `已同步`
6. 选择图片文件夹并开始上传

## 手动模式

如果客户电脑不方便装桥接脚本，也可以在本地上传器的高级项里手动填写：

- `token`
- `factoryId`
- `masterFactoryId`
- `laravel_session`（可选）

保存后一样可以上传。
